# agasi documentation sources

These **Markdown** files are the source for the **VitePress** site.

- **Develop:** from `packages/agasi`, run `npm run docs:dev`.
- **Build:** `npm run docs:build` → static output in **`.vitepress/dist/`** (gitignored).
- **Search:** the default theme’s **local** provider indexes all pages at build time; the published site needs no search API.

Configuration: **`.vitepress/config.mts`**.
